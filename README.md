# Rock-Paper-Scissors-Game
A simple console-based Rock-Paper-Scissors Game implemented in JavaScript.
This project is part of the *Javascript Basics* section of [The Odin Project](https://www.theodinproject.com/lessons/foundations-rock-paper-scissors) aimed to solidify my understanding fundamental JavaScript concepts.

## Learning Goals
- Variables, operators and data types
- Conditional statements
- functions
- problem solving

## How to Play
1. Run the game in the console.
2. Enter your choice: Rock, paper or Scissors through  a prompt.
3. Computer generates it's own move and the results are displayed in the console after each round.
4. After 5 rounds, the game declares the overall winner.
